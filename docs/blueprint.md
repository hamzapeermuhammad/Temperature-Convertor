# **App Name**: TempConvert

## Core Features:

- Temperature Input: Accept numerical input for the temperature value from the user via a number input field.
- Unit Selection: Provide a dropdown selector to choose the input unit (Celsius, Fahrenheit, or Kelvin).
- Unit Conversion: Calculate the converted temperature values for the other two units based on the input.
- Output Display: Display the converted temperature values for each of the other units in a clear, readable format.
- Error Handling: Handle invalid input gracefully, displaying error messages as needed, ensuring users provide valid numerical values for temperature.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) evoking coolness and temperature awareness.
- Background color: Very light blue (#F0F8FF), almost white, for a clean and airy feel.
- Accent color: Pale yellow (#FFFFE0), an analogous hue to suggest sunlight and warmth. Use sparingly for CTAs and key elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Use simple, clean icons representing Celsius, Fahrenheit, and Kelvin next to their respective display fields.
- A clean and straightforward layout with a prominent input field and clearly labeled output fields.
- Subtle transitions when converting temperatures to provide a smooth user experience.
